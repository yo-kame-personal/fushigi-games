# ふしぎゲームズ５

「シンプルだけど、奥が深い」をテーマにしたオリジナルミニゲーム5本セット。
ブラウザだけで動きます（インストール・ビルド不要）。

## 起動方法

`index.html` をダブルクリックするだけ。
またはターミナルから:

```bash
open ~/Desktop/dev/fushigi-games/index.html
```

ゲーム中は **ESC** キーでいつでもメニューに戻れます。

## 収録ゲーム

### 🎨 いろかさね
- **ルール**: シアン・マゼンタ・イエローのインクを「行」か「列」まるごとに重ねて、みほんと同じ盤面を作る。
- **操作**: マウスクリック（インク切り替えはキー 1 / 2 / 3 でもOK）。
- **奥深さ**: インクはON/OFF式（同じ色を2回重ねると消える）。行と列の重なりをどう使うかが鍵。手数の最小化に挑戦しよう。

### 👻 じぶんエコー
- **ルール**: ★を取るたび「1.5秒前の自分」をなぞるゴーストが増える。ゴーストに触れたら負け。
- **操作**: 矢印キー / WASD で移動。
- **奥深さ**: 敵は全部「過去の自分の動き」。未来の自分の邪魔にならない動き方を考える必要がある。止まると追いつかれる！

### 🔢 かずつなぎ
- **ルール**: となり合う数字（ななめOK）を「差が1」または「2倍/半分」のルールでなぞってつなぐ。3つ以上で消える。
- **操作**: マウスドラッグ。
- **奥深さ**: 得点はチェーン長の2乗 × 10。長くつなぐと時間ボーナスもあるので、欲張りと安全のバランスが問われる。

### 🪐 ひっぱりスター
- **ルール**: クリック長押し中だけ、ボールがマウスに引き寄せられる。リングをくぐるとスコア。トゲ玉に当たったら負け。
- **操作**: マウス長押しのみ（1ボタン）。
- **奥深さ**: 直接は操作できず「引力」でしか動かせない。慣性と壁バウンドを読んだ立ち回りが上達ポイント。

### 💥 ふくらみパルス
- **ルール**: 自分は勝手にふくらみ、大きいほどスコアが速く増える。トゲが飛んでくる前に警告線が出る。クリックで一瞬だけ小さくなれる。
- **操作**: クリック or スペースのみ（1ボタン)。
- **奥深さ**: 「大きいまま耐える＝ハイリスク・ハイリターン」。縮むタイミングの見極めがすべて。

## 技術構成

- HTML + CSS + JavaScript（Canvas API）の **1ファイル構成**（外部ライブラリなし）
- ベストスコアはブラウザの `localStorage` に自動保存
- 効果音は Web Audio API で生成（音声ファイル不要）
